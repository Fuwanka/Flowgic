# Flowgic
Flowgic (Стартап-проект по Акселерационной программе "Пятый элемент") - MiniCRM система для автоматизации бизнес процессов логистических предприятий и организации более удобного взаимодействия с клиентами.

# ВАЖНО ДЛЯ ВСЕХ
Порядок выполнения команд:

1. **Создать виртуальное окружение**:
~~~bash
python -m venv venv
~~~
---
2. **Активировать виртуальное окружение**:
~~~bash
venv\Scripts\activate
~~~
---
3. **Установить зависимости**:
~~~bash
pip install -r requirements.txt
~~~
---
4. **Применить миграции**:
~~~bash
python src\manage.py makemigrations
~~~
~~~bash
python src\manage.py migrate
~~~
---
